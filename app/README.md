# HighTable demo app

This is a demo app around [HighTable](https://github.com/hyparam/hightable) used to perform benchmarks.

## Build

```bash
npm i
npm run build
```

The build artifacts will be stored in the `dist/` directory and can be served using any static server, eg. `http-server`:

```bash
npm i -g http-server
http-server dist/
```
